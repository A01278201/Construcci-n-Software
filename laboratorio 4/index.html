Preguntas

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?

    Ambos son lenguajes de programación, pero Java es de propósito general, compilado y orientado a objetos, mientras que JavaScript es un lenguaje interpretado y se usa principalmente en el desarrollo web para la manipulación del DOM. Aunque comparten parte del nombre, son muy distintos en su estructura y uso.

  2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5)

    Algunos métodos del objeto Date son: getDate(), getMonth(), getFullYear(), getHours(), toISOString().

  3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5)

    Los arreglos tienen métodos como: push(), pop(), slice(), map(), filter().

  4. ¿Cómo se declara una variable con alcance local dentro de una función?

    Se declara utilizando let o const dentro de la función, por ejemplo: function ejemplo() { let x = 5; }.

  5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    Las variables globales pueden ser modificadas desde cualquier parte del código, lo que puede causar errores difíciles de detectar o conflictos si varias funciones las usan sin control. También hacen que el código sea menos modular.

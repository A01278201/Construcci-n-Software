Ejercicio 1: Tabla de potencias

Este ejercicio pide un número y muestra una tabla con los números del 1 hasta ese número, junto con sus cuadrados y cubos.

Ejercicio 2: Suma aleatoria

Este ejercicio genera dos números aleatorios, pide al usuario su suma, y muestra si fue correcta junto con el tiempo que tardó en responder.

Ejercicio 3: Contador de números

Este ejercicio analiza una lista de números ingresada por el usuario y cuenta cuántos son negativos, ceros y positivos.

Ejercicio 4: Promedios por fila

Este ejercicio permite ingresar una matriz de números y calcula el promedio de cada fila.

Ejercicio 5: Número inverso

Este ejercicio toma un número e invierte el orden de sus dígitos.

Ejercicio 6: Radio de un círculo

Este ejercicio calcula el área y perímetro de un círculo a partir del radio ingresado en centímetros.
